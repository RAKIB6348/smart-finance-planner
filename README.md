# Smart Finance Planner

Smart Finance Planner is a small vanilla JavaScript budget tracker. It uses plain HTML, CSS, and DOM manipulation without any framework.

## What It Does

- Sets a monthly budget.
- Creates custom expense categories.
- Adds expenses with a name, amount, and category.
- Calculates total expenses and remaining balance automatically.
- Shows budget usage with a dynamic progress bar.
- Displays recent expense history.
- Deletes individual expenses.
- Shows user-friendly success and error messages.
- Clears all expenses without removing the budget or categories.
- Resets the whole planner back to default data.
- Saves the budget, categories, and expense list in browser `localStorage`, so data remains after refreshing the page.

## Files

- `index.html` contains the page structure.
- `style.css` contains the responsive dashboard design.
- `app.js` contains the DOM logic, calculations, local storage, and expense list rendering.

## How To Run

Open `index.html` in a browser. No build step or package installation is required.
